基于Hexo在Github上创建Blog的流程

下载nodejs，完成后win+R，命令行输入node -v检查是否成功安装

注册一个Github账户
点击create a new repository以创建一个仓库用以储存Blog所需文件，其中仓库名推荐格式为（用户名）.github.io,下方选项Add.gitignore和Add a license建议勾选None，同时也推荐创建README文档，然后点击 create repository

安装git for wins，完成后win+R，命令行输入git检查是否成功安装

在下载的git文件夹中找到git-bash.exe并运行，在弹窗中输入ssh-keygen -t rsa -C “你的QQ邮箱地址”，随后一直回车直到弹窗中的程序运行完毕

登录Github,点击用户头像，在侧边栏找到“SSH and GPG keys”,添加SSH密钥，title推荐填写用户名，密钥在C盘–>用户–>Administrator–>.ssh文件夹，将其中的id_rsa.pub文件用记事本打开，复制即可

打开Hexo官网，在合适的地方新建一个文件夹，win+R命令行输入cd 文件夹路径 跳转，随后输入npm install hexo-cli -g并回车。下载完毕后输入hexo init 博客名，随后cd 博客名 跳转，输入npm install并回车。完成后输入hexo s -p 5555并回车，浏览器访问localhost：5555可以本地浏览已经创建的Hexo博客

下载Sublime，完成后运行sublime_text.exe,将博客文件夹拖入打开的sublime窗口，在侧边栏打开_config.yml文件，完成以下修改：deploy: (空)
type: git ， repo: https://github.com/用户名/仓库名.git ， branch: master ，url：仓库名

win+R命令行输入npm install hexo-deployer-git –save下载插件，随后输入hexo clean && hexo g && hexo d部署文件，不放心的话可以先输入hexo clean && hexo g && hexo s本地浏览，随后输入git config –global user.email “你的QQ邮箱地址”,以及git config –global user.name “用户名”，在弹窗中登录Github，之后就可以在浏览器中输入https://仓库名访问博客了
